注：此demo只包含nodejs接口部分内容，不含前端页面相关代码逻辑
1、下载demo文件
    git clone 
2、安装依赖
    npm install
3、安装netlify cli（nodejs版本需在18.14.0及以上）（已安装过请忽略）
    npm install netlify-cli -g
4、初始化netlify项目
    netlify init
        选择  Create & configure a new site
        依次选择并回车可以看到生成的 Admin URL、URL、Site ID，会自动link到新的netlify项目
